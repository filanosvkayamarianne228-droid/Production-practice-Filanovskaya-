# Система учета заявок IT-отдела
## Описание проекта
Консольное приложение для автоматизации процесса обработки и учета заявок в IT-отделе 
учебного заведения "Колледжа туризма и прикладных технологий". Программа заменяет неорганизованный приём заявок по электронной почте и устно на 
централизованную систему, исключая потери обращений и обеспечивая полный контроль выполнения.

## Установка и разработка программы 
1. Клонировать репозиторий
2. Открыть в Visual Studio
3. Собрать проект

## Использование
* Создание заявок
* Просмотр списка
* Изменение статуса
* Поиск по параметрам
